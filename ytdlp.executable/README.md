# YTDLP-Executable  

**YTDLP-Executable** is a NuGet package that includes the latest version of `yt-dlp.exe` (v2025.03.21). It is designed to complement the **YTDLP-Wrapper** library or any other application that requires the `yt-dlp` executable for powerful media downloading capabilities.  

## Features  
- **Latest Version**: Always ships with the latest version of `yt-dlp.exe` (current version: **v2025.03.21**).  
- **Standalone Package**: Provides only the `yt-dlp.exe` binary for easy integration.  
- **Complementary**: Works perfectly with the **YTDLP-Wrapper** library.
